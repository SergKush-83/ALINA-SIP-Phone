# ALINA-SIP-Phone
библиотека SIP-телефонии для ESP32 с поддержкой Ethernet (WT32-ETH01).    \## Особенности    \- Полная реализация SIP протокола (RFC 3261)  \- Поддержка RTP/RTCP для аудио  \- Веб-интерфейс для конфигурации  \- Мониторинг системы и автоматическое восстановление  \- Поддержка нескольких одновременных вызовов  \- Аутентификация Digest MD5
